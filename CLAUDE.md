# 인쇄용지 경쟁 분석 대시보드 - 프로젝트 컨텍스트

## 프로젝트 개요
인쇄용지 시장 경쟁사 분석 대시보드 (단일 페이지 웹앱)
- `index.html` — 메인 앱 (Tailwind CSS, Chart.js, Leaflet.js)
- `data.js` — 모든 데이터 정의 (JSON 형태)
- 브라우저에서 직접 열어서 사용 (빌드 없음)

## 탭 구성
1. 수출실적 — 중국산 수출 실적 차트/테이블
2. 가격동향 — 중국 내수가 및 수출가 추이
3. 마켓맵 — Leaflet 지도 기반 공장 위치
4. 경쟁사 — 경쟁사 프로필 카드
5. 뉴스 — 뉴스피드

## 데이터 파일 (JSON, 수동 업데이트)
- `260325_뉴스피드.json`
- `260325_경쟁사프로필.json`
- `260325_공장CAPA.json`
- `260325_중국내수가격.json`
- `260325_중국산수출실적.json`
- `260325_중국주요지표.json`

## 기술 스택
- 순수 HTML/CSS/JS (프레임워크 없음)
- Tailwind CSS (CDN)
- Chart.js 4.4.7
- Leaflet.js 1.9.4
- Noto Sans KR 폰트

## 개발 규칙
- 빌드 단계 없음 — 브라우저에서 직접 열기
- 데이터는 `data.js`에서 관리
- 스타일은 인라인 CSS 클래스 사용 (index.html 상단 `<style>` 블록)
- 한국어 UI
