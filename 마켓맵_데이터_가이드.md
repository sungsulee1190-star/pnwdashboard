# 마켓맵 국가별 데이터 정리 가이드

## 핵심 원칙
**하나의 국가 = 하나의 블록**, 그 안에 플레이어들을 나열합니다.  
플레이어 타입: `mill` (제지사) | `distributor` (유통사) | `printer` (인쇄소/최종 사용처)

---

## 엑셀 템플릿 구조 (권장)

| 컬럼명 | 예시 | 설명 |
|--------|------|------|
| country | Australia | 국가명 (영문) |
| country_kr | 호주 | 국가명 (한국어) |
| player_name | PaperlinX | 업체명 |
| player_type | distributor | mill / distributor / printer |
| market_share_pct | 35 | 시장 점유율 (%) |
| annual_usage_mt | 12000 | 연간 사용량 (MT) |
| partner_company | 한솔 | 한국 파트너 (한솔/무림/한국/없음) |
| partner_status | 현재 | 현재/과거/협상중 |
| notes | 최대 유통사, 2개 창고 운영 | 비고 |
| data_year | 2025 | 데이터 기준 연도 |

## 데이터가 너무 개판인 경우 — 현실적 접근

원시 데이터가 정리되지 않은 상태라면:

1. **최소 단위로 시작**: 국가명 + 주요 파트너사 + 예상 물량만
2. **확실한 것만**: 불확실한 데이터는 `notes`에 "추정" 표기
3. **점진적 업데이트**: 출장/미팅 후 아는 정보 그때그때 추가
4. **시각화 우선**: 완벽하지 않아도 지도에 올리면 패턴이 보임

---

## 시각화 방향 (구현 예정)

- **지도 레이어**: 국가별 원 크기 = 시장 규모, 색상 = 한국산 점유율
- **클릭 팝업**: 해당 국가의 플레이어 리스트 + 파트너십 현황
- **네트워크 뷰**: 한국 3사 ↔ 국가별 파트너 연결선
- **필터**: 타입별(제지사/유통사/인쇄소), 파트너별
